## Installation

Use the node package manager [npm](https://docs.npmjs.com/cli/v8/commands/npm-install) to install dependencies.

```bash
npm install
```

## Usage

Compile contracts:

```bash
npm run compile
```

Deploy contracts

```bash
npm run deploy
```

Run tests

```bash
npm run test
```

Start hardhat node for the external use

```bash
npm run node
```
